What is ACE?

ACE, the Amortized Conditioning Engine, is a transformer-based probabilistic model. It is trained on many simulated problem instances so that, at runtime, one forward pass can return probability distributions instead of solving a new inference problem from scratch. ACE was developed by the Machine and Human Intelligence research group at the University of Helsinki and collaborators; see the ACE project page.

It sits in the same broad family as Prior-Fitted Networks (PFNs) and Transformer Neural Processes (TNPs): models that learn how to condition on examples. ACE extends that idea by treating more than data as tokens. Observations, unknown queries, interpretable latent variables, and runtime prior information can all appear in the same conditioning set.

That means ACE can condition on data, known parameter values, or a user's prior belief, and it can predict new data as well as quantities like model parameters, class labels, or the location of an optimum. The examples here show those token-level conditioning moves in small, inspectable tasks using small pretrained networks that run in your browser.
